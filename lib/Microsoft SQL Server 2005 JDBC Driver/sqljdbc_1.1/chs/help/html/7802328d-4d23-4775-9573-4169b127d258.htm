Microsoft SQL Server 2005 JDBC 驱动程序支持在将 SQL Server 用作其数据库的 Java 应用程序中使用 JDBC 基本数据类型和高级数据类型。

对 SQL Server 数据类型和 Java 语言类型以及对象之间的转换而言，JDBC 类型系统可起到中介作用。JDBC 类型是以 SQL-92 类型和 SQL-99 类型为模型而建立的。JDBC 驱动程序遵守 JDBC 规范，用于在可预见性和灵活性之间达成适当的平衡。

本节中的主题说明如何使用基本数据类型和高级数据类型，以及如何将数据类型转换成其他数据类型。

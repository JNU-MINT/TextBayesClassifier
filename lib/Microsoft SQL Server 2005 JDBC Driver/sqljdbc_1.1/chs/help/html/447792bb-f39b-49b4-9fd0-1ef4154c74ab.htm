若要使用 Microsoft SQL Server 2005 JDBC Driver 访问 SQL Server 数据库中的数据，计算机上必须安装有下列组件：

  • Microsoft SQL Server 2005 JDBC Driver

  • Java 运行时环境

Java 运行时环境要求

JDBC 3.0 功能要求具有 1.4 版或更高版本的 Java 运行时环境 (JRE)，因此 JDBC 驱动程序也需要 1.4 版或更高版本的 JRE。JDBC 驱动程序可在所有主要的虚拟机中运行，但在 Sun JRE 1.4 或更高版本上可得到正式支持：

SQL Server 要求

JDBC 驱动程序专为使用 SQL Server 2005 引入的功能而设计，但它也可以向后兼容 SQL Server 2000（包括 64 位版本）。

操作系统要求

JDBC 驱动程序可在任何支持使用 Java 虚拟机 (JVM) 的操作系统上工作。但是，仅 Sun Solaris、Red Hat Linux 和 Windows 2000 或更高版本的操作系统对其提供正式支持。

支持的语言

JDBC 驱动程序支持所有的 SQL Server 列排序。

有关排序的详细详细，请参阅 SQL Server 联机丛书中的“处理排序”。

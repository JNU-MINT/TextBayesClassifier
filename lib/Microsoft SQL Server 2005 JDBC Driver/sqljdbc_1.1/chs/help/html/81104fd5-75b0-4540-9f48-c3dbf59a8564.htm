Creates a SQLServerCallableStatement object that generates SQLServerResultSet objects with the given type, concurrency, and holdability.

public java.sql.CallableStatement prepareCall(java.lang.String sql,
                                              int nType,
                                              int nConcur,
                                              int nHold)

参数

sql

A string containing an SQL statement.

nType

An integer that indicates the result set type.

nConcur

An integer that indicates the result set concurrency type.

nHold

An integer that indicates the result set holdability.

返回值

A CallableStatement object.

异常

SQLServerException

备注

This prepareCall method is specified by the prepareCall method in the java.sql.Connection interface.
